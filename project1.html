Project 1

Introduction

Recently, I was introduced to the basics of CAD (Computer-Aided Design) and CAM (Computer-Aided Manufacturing) as they relate to 3D printing. I learned how to use the Prusa MINI+ 3D printer and Prusa Slicer software, including configuring the slicer and downloading 3D print files. I explored how to slice a 3D file and prepared it for 3D printing (CAM), transforming digital design to physical objects. I also ran a more difficult “torture test” to learn more about the limitations of the 3D printer.

Preparation Before Printing

  1. 1. Download Prusa Slicer and Prusa MINI+.
  2. 2. Download the 3D model file you want to print and edit it in Prusa Slicer. Adjust the print settings, including layer height, infill percentage, and more.
  3. 3. Slice the file. “Slicing” refers to the process of converting a digital 3D model into a set of instructions (called G-code) that the 3D printer can understand.
  4. 4. Export the G-code into the USB drive formatted for the printer. Double-check that the file is saved with the correct extension and named appropriately for easy identification.
  5. 5. Insert the USB into the printer and navigate through the printer’s menu to locate the file.
  6. 6. Start the print and monitor the first few layers to ensure proper adhesion to the print bed. Make adjustments if necessary, such as leveling the bed or cleaning the nozzle.


First Print: Name Tag

For my first print, using the Prusa MINI+ printer, I wanted to create a simpler project to explore its capabilities. I successfully created a 3D-printed name tag with my name, “Aidan Wang-Fan,” and achieved a crisp, clean color transition for the text. The process was pretty smooth, and I was impressed by how easily Prusa Slicer worked with the printer. The color change feature worked perfectly, adding a clean, professional-looking contrast to the text. However, the final print showed some sign of stringing. Stringing, also known as oozing, happens when small strings of filament are left behind as the extruder moves between different parts of the print. This might’ve happened because the extruder temperature was too high. For this project, we used PLA and printed at 215°C. At higher temperatures, the plastic is more likely to leak out during movement. Despite these minor setbacks, the project demonstrated the efficiency and reliability of the Prusa ecosystem, resulting in a visually appealing nametag.

CAM Settings for First Print:

  • Used Filament (g): 4.03
  • Filament: Prusament PLA
  • Printing Time: 22 minutes
  • Layer Height: 0.2mm
  • Infill Percentage: 15%


Torture Test

For my second 3D print (torture test), I selected a complex STL file designed to push the limits of the printer. The model included intricate details, overhangs, and thin structures, which tested the printer’s capability to handle challenging geometries.

During the print, several issues arose, including poor adhesion on the finer details, stringing, and failure in printing the right arm cleanly. The right arm experienced deformation and an uneven surface. Through my knowledge of the 3D printer, I assume that the problems were likely caused by both support and cooling.

1. Supports

The failure in the right arm was primarily due to insufficient support. Overhangs in the model, such as the outstretched arm and other unsupported sections, lacked stability during the print. Without adequate support, the filament had no base to adhere to, leading to sagging, uneven layers, and a rough finish.

2. Cooling

Cooling also played a role, particularly for the right arm. For small and thin structures, like the arm and fingers, the layers did not have enough time to cool and solidify before the printer moved on to the next layer. This resulted in soft, warped layers. In addition, the lack of supports worsened the deformation.

3MF vs STL

Finally, comparing the 3MF file (used for slicing) to the STL file (a basic design file), I learned that the key difference between these two file types is that 3MF files can store additional data, including material, color, texture, orientation, supports, and printing parameters, whereas STL files cannot. As a result, the name tag print file automatically incorporated the color change, but an STL file would not be able to do so.

CAM Settings for Second Print:

  • Used Filament (g): 11.90
  • Filament: Prusament PLA
  • Printing Time: 2 Hours and 52 Minutes
  • Layer Height: 0.3mm
  • Infill Percentage: 15%

Reflection

Both prints taught me a lot about 3D printing. The name tag was a great way to get the hang of slicing and printing, while the torture test really showed me how tricky it can be to print more complex designs. Looking ahead, I plan to tweak my settings, especially for supports and cooling, to improve the overall print quality.

With each project, I'm feeling more confident using the Prusa MINI+ and getting better at the whole process.

I’m really excited to keep experimenting with new designs and learn even more about 3D printing along the way!
