Project 4

Introduction

Recently, I was introduced to open-source assistive technology and its role in 3D printing. For this project, I worked with a partner to create a 150% scale e-NABLE prosthetic hand using a Prusa Mini+ 3D printer and Fusion 360. I explored resources from the e-NABLE hub and the NIH 3D Print Exchange to guide the printing and assembly process. Along the way, I learned about the challenges of building functional prosthetic devices, the importance of clear documentation, and how rapid prototyping can improve assistive technology.

Preparation Before Printing

  1. 1. Download the different hand part (STL files) off of e-NABLE and upload into Prusa Slicer.
  2. 2. Scale all the different file to 150%.
  3. 3. Fit all the different hand parts using 3 build plates.
  4. 4. Export the G-code into the USB drive formatted for the printer. Double-check that the file is saved with the correct extension and named appropriately for easy identification.
  5. 5. Insert the USB into the printer and navigate through the printer’s menu to locate the file.
  6. 6. Start the print and monitor the first few layers to ensure proper adhesion to the print bed. Make adjustments if necessary, such as leveling the bed or cleaning the nozzle.

Reflection on the Hand Print

At first, I was able to fit all the parts onto two build plates, but then I realized we had to scale everything to 150%. This was important so that everyone could swap parts and assembling them would be easier. Because of this, we had to reorganize everything, and in the end, it took three build plates instead of two.

One of the biggest issues we ran into was with the pins for the hands. Since they had such a small surface touching the build plate, they didn’t stick well. To fix this, we added three layers of rafts to make sure they stayed in place. We also used organic supports, but only on the build plate, to help support the fingers while printing.

When the prints were finished, I didn’t notice any major problems, but the supports were still really hard to remove, even though they were organic supports. To make things easier in the future, the instructions could include more details on how to scale and organize parts efficiently. It would also help to have better guidelines on using supports and rafts, as well as some tips for removing supports without damaging the parts.

Assembling the Hand and Reflection

To assemble the hand, we used this PDF file, which we found very helpful. We relied a ton on the pictures.

The assembly process went smoothly overall, but we ran into a few challenges. One issue was figuring out which pins to use. Since there were no clear instructions, we had to test each pin until we found the right fit, which took extra time. Labeling the pins in the instructions would make this step much easier.

Another challenge was bending the “thermo gauntlet,” the piece that covers the arm. We initially thought it was a simple bend, but when we tried to attach it to the hand shell, the holes didn’t align. We had to reheat and reshape it multiple times to get the correct fit.

Tying the strings to the fingers was also tricky. The instructions were helpful, but I originally thought we needed five separate strings—one for each finger. It turned out that only two long strings were needed to control four fingers, plus one for the thumb. Since we had already cut them into four strands, we had to tie them back together. We watched a video tutorial for guidance.

Lastly, we didn’t use the palm mesh because we wanted to fit our hand inside, so we attached a handle instead. However, for those without hands, the palm mesh would be necessary and would need to be heated to create a curve. Despite these small issues, the assembly went well overall.

Extend, critique & reflect

For the next part of the project, we were asked to add a modification to our design. At this point, we wanted to design something fun and unique to modify the e-NABLE hand. After brainstorming a few ideas, we decided to add Wolverine claws to the prosthetic. My partner was immediately on board with the idea, so we proceeded with designing it. I began by sketching out the basic concept of the claw and how it would integrate with the hand.

The design starts with a base that would be securely fixed to the e-NABLE hand. I incorporated a hole in the base, allowing it to be screwed into place. On top of the base, I added three half-circular shapes, each with a 10mm hole. These holes would serve as the pivot points for a hinge mechanism that would allow the claw to move. The claw itself would be divided into three movable parts, each controlled by the hinge system.

This initial design is just a starting point, and I’m aware there are areas that need further refinement. For example, in my sketch, I drew the claw in a tense position, but I’ll need to smooth out the mechanism to ensure the claws can extend and retract seamlessly. There’s still work to be done to ensure the claw functions smoothly and safely, but I’m excited to continue refining the design.
